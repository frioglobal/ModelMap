# ModelMap

A toolkit for Model Mapping experimentation.

Tested with python 3.6.8

Dependencies:
scikit-learn 0.19.1
pandas 0.22.0
matplotlib 2.2.2
numpy 1.15.4
gpy 1.9.2


To run the experiments, launch them from the toolkit root folder.

Example: ```python TPC-C-Scenario1.py```
